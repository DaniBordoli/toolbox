
## Instalación

=> Clona el repositorio:
   git clone https://github.com/tuusuario/toolbox-app.git

# Backend

1. cd backend
2. npm install
3. npm start

Pruebas
npm test

Versiones de Paquetes Utilizados

Node.js: ^14.17.6
Express: ^4.17.1
Axios: ^1.5.0
Cors: ^2.8.5
Mocha: ^10.2.0
Chai: ^4.3.8
Standard: ^17.1.0

El servidor estará disponible en http://localhost:3000

3. npm start
